Contexto

InmoSpec es un proyecto de analisis de datos inmobiliarios orientado a estudiar la evolucion del precio de la vivienda en Espana y detectar patrones geograficos y tendencias del mercado.

El flujo comienza con scraping de anuncios de Fotocasa mediante Selenium, continua con limpieza y transformacion de datos en Pandas, y escala el procesamiento con Spark sobre una arquitectura Hadoop con HDFS desplegada en una instancia AWS EC2.

Pipeline de datos

  • Extraccion automatizada de datos de Fotocasa con Selenium.
  • Limpieza, normalizacion y enriquecimiento del dataset con Pandas.
  • Procesamiento distribuido con Apache Spark para agregaciones y calculos intensivos.
  • Persistencia de datos en HDFS dentro de un entorno Hadoop.
  • Ejecucion integral en una maquina AWS EC2 para centralizar el flujo end-to-end.
  • Construccion de informes y visualizaciones interactivas en Power BI.
